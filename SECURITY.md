# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** report it in a public issue.

Submit it privately via GitHub Security Advisory (Security → Advisories).

We will acknowledge receipt within 48 hours and work on a fix as soon as possible.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |
| older   | ❌        |
